Homework 1

Research

Questions

  1. What is the definition of statistics?
  2. What is a dataset? From the observation units to the dataset. Attributes and variables.
  3. Find interesting application of statistics in Cybersecurity.

Answers

  1. Statistics is the science that, thanks also to methods and mathematics tools, deals with the information acquisition about a phenomenon and how these information are organized, analysed and interpreted.

  2. A dataset is a collection of descriptions of different instances of the same phenomenon. These descriptions could take a variety of forms, but it is important that they are descriptions of the same thing. For example, my grandfather collected the daily rainfall in his garder for many years; we could collect the height of each person in a room.
     An observation unit, sometimes also called statistical unit, is the single inhabitans of the population over which is performed an observation. In other words, the unit of observation is the unit at or for which data is collected. At the end, after choosing a phenomenon to observe, choosing observation units and collecting descriptions, we have a dataset.

    An attribute is a quality of an object (person, thing, etc.). As an example for a person, the possible attributes are: gender, age, sports etc. A variable is the set of all possible value that an attribute can take.

    • A very common way in which Statistics is used in Cybersecurity is to assess trends in order to take countermeasures. As an example, in the following link is showed how much UK organisations deal with Cybersecurity; the reasearch has been done by the UK government
      https://assets.publishing.service.gov.uk/government/uploads/system/uploads/attachment_data/file/987652/CSBS_2021_business_trends_infographic.pdf
    • The purpose of the following paper is to describe the development of the cybersecurity datasets used to train the algorithms which are used for building IDS detection models, as well as analyzing and summarizing the different and famous Internet of things (IoT) attacks.
      https://www.researchgate.net/profile/Anton-Konev-2/publication/358223470_The_Comparison_of_Cybersecurity_Datasets/links/61f6d421007fb50447260c47/The-Comparison-of-Cybersecurity-Datasets.pdf
    • Another important way in which Statistics is used in Cybersecurity, is to discover if something dangerous is happening in our incoming network. The following paper illustrates a statistical analysis of network data. https://www.researchgate.net/publication/261581408_Statistical_Analysis_of_Network_Data_for_Cybersecurity

Questions

  1. What are the main differences between C# and VB.NET?

Answers

  1. The main differences between these two languages are:
    • syntattical; e.g. C# requires the semicolon at the end of the instructions, meanwhile VB.NET does not;
    • the fact that VB.NET has an higher level of abstraction with respect of C#; e.g. in VB.NET the following declaration Dim i As Integer = "1" automatically converts a String to an Integer (those are two totally different types);
    • the implementation of interfaces;
    • the raiseing of events;
    • the handling of events; e.g. is possible, in VB.NET, to connect event handlers to object variables rahter than to objects;

Research on app

Questions

  1. Summarize the main aspects the of the CSV protocol RFC 4180 (definition and rules).

Answers

  1. A comma-separated values (CSV) file is a text file that uses a comma to separate values. A CSV file stores tabular data, such as numbers or text in a spreadsheet, in plain text. Each line of the file is a data record. Each record consists of one or more fields, separated by commas. Thus, the file format gets its name from using the comma to separate different values. Example where each record is separated by a new line (CRLF):
    value1,value2,value3CRLF
    value4,value5,value6CRLF

     There maybe an optional header line appearing as the first line of the file with the same format as normal record lines. This header will contain names corresponding to the fields in the file and should contain the same number of fields as the records in the rest of the file. Each field may or may not be enclosed in double quotes.
    Example:
    "age","heigth"CRLF
    10,130CRLF
    51,180
